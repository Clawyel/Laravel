NOT:

Api fonksiyonlarına erişmek için postman kullanıldı
CSRF-TOKEN bilgisini eklemek için app.layout.blade dosyasına tag'i içerisine bu bilgiyi eklememiz gerekiyor.
Postman kullanabilmek için yeni bir header oluşturum X-CSRF-TOKEN değerine sunucudaki herhangi bir sayfadan alacağınız Token bilgisini eklemeniz gerekiyor.
Postman içeriisnde daha doğru sonuçlar alabilmek için headers değerlerine Accept:application/json değerini eklemelisiniz.
Son Olarak Seeder Özelliğinden faydalanarak veritabanı oluşturup tablolara default değerleri ekletebilirsiniz.

Kayıt İşlemi:

Api fonksiyonlarında veri işlemleri yapabilmek için önce sisteme kayıt olmanız gerekiyor


url -> /registerUser
method -> post
gerekli parametreler name,email,password

Login İşlemi:

Kayıt olduktan sonra email ve şifre kullanılarak login olmalısınız
Json Formatında aldığınız cevap içinden api_token bilgisini kopyalayın
Postman içerisine yeni header ekleyin Authorization: Bearer kopyaladığınız-token-değeri
Bu işlemlerden sonra api içeriisndeki verilere erişebilirsiniz


url -> /loginUser
method -> post
gerekli parametreler email,password donüş değeri : user json data


url -> /authSys/getUser
method -> get
parametre yok donüş değeri : auth user json data
